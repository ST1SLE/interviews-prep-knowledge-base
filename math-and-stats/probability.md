## Q1. "Что такое случайная величина? Дискретная vs непрерывная"

> "Случайная величина — функция, которая сопоставляет результату эксперимента числовое значение. **Дискретная** — принимает конечное или счётное множество значений (кол-во дефолтов). **Непрерывная** — любое значение из интервала (доход клиента). Для дискретной — PMF (probability mass function), для непрерывной — PDF (probability density function). В ML: метки классов — дискретная, предсказание регрессии — непрерывная."

## Q2. "Основные распределения — какие знаешь?"

| Распределение | Тип | Когда встречается | Параметры |
|---------------|-----|-------------------|-----------|
| **Bernoulli** | Дискретное | Одно событие: да/нет (дефолт кредита) | $p$ |
| **Binomial** | Дискретное | $n$ независимых Bernoulli (сколько дефолтов из 100 кредитов) | $n, p$ |
| **Poisson** | Дискретное | Редкие события за период (кол-во жалоб в день) | $\lambda$ |
| **Uniform** | Непрерывное | Всё равновероятно (random seed) | $a, b$ |
| **Normal (Gaussian)** | Непрерывное | Рост, вес, ошибки модели | $\mu, \sigma^2$ |
| **Exponential** | Непрерывное | Время до события (время между покупками) | $\lambda$ |

> "В ML нормальное распределение — king. Многие алгоритмы предполагают нормальность: Linear Regression, Gaussian Naive Bayes, инициализация весов нейросетей. Если данные не нормальны — применяем log-transform, Box-Cox."

## Q3. "Мат. ожидание и дисперсия — что это и зачем?"

> "$E[X]$ (мат. ожидание) — 'центр масс' распределения, средневзвешенное значение. Для дискретной: $E[X] = \sum x_i \cdot P(x_i)$. Для непрерывной: $E[X] = \int x \cdot f(x) \, dx$.
>
> $\text{Var}[X]$ (дисперсия) — мера разброса. $\text{Var}[X] = E[(X - E[X])^2] = E[X^2] - (E[X])^2$. $\sigma = \sqrt{\text{Var}}$ — стандартное отклонение, в тех же единицах, что и $X$.
>
> В ML: bias модели $\approx E[\text{ошибка}]$, variance модели $\approx \text{Var}[\text{ошибка}]$. Bias-variance tradeoff — фундамент ML."

**Свойства:**

$$E[aX + b] = a \cdot E[X] + b$$

$$\text{Var}[aX + b] = a^2 \cdot \text{Var}[X]$$

$$\text{Var}[X + Y] = \text{Var}[X] + \text{Var}[Y] \quad \text{(если X, Y независимы)}$$

## Q5. "Теорема Байеса — формула и интуиция"

> $$P(A \mid B) = \frac{P(B \mid A) \cdot P(A)}{P(B)}$$
>
> Интуиция: обновляем наши убеждения при получении новых данных.
> - $P(A)$ — prior (до наблюдений, наше начальное убеждение)
> - $P(B \mid A)$ — likelihood (вероятность увидеть данные при условии гипотезы)
> - $P(A \mid B)$ — posterior (обновлённое убеждение после данных)
>
> Пример: вероятность дефолта клиента. Prior: 5% клиентов делают дефолт. Likelihood: если клиент задерживал платежи, $P(\text{задержки} \mid \text{дефолт}) = 0.8$. Posterior: $P(\text{дефолт} \mid \text{задержки})$ сильно возрастает.
>
> В ML: Naive Bayes классификатор, MAP estimation, Bayesian сети (использовал при подготовке к SberQ)."

## Q13. "Закон больших чисел — зачем в ML?"

> "**LLN**: при увеличении размера выборки выборочное среднее сходится к истинному математическому ожиданию. $\bar{X}_n \to E[X]$ при $n \to \infty$.
>
> Применение в ML:
> 1. **Training loss** — среднее по батчам приближает истинную ожидаемую ошибку
> 2. **Cross-validation** — среднее по K фолдам → стабильная оценка качества
> 3. **Ensemble** — среднее предсказаний K моделей → менее шумное предсказание (variance ↓)
> 4. **Monte Carlo методы** — оценка сложных интегралов через семплирование"

## Q14. "Условная вероятность и независимость"

> $P(A \mid B) = \frac{P(A \cap B)}{P(B)}$ — вероятность A при условии, что B произошло.
>
> **Независимость**: A и B независимы, если $P(A \mid B) = P(A)$, т.е. знание B не меняет вероятность A. Эквивалентно: $P(A \cap B) = P(A) \cdot P(B)$.
>
> В Naive Bayes: предполагаем условную независимость фичей при данном классе — $P(x_1, x_2 \mid y) = P(x_1 \mid y) \cdot P(x_2 \mid y)$. На практике это не выполняется, но модель всё равно работает хорошо для текстов (спам-фильтры)."

## Q15. "Метод обратной функции распределения (Inverse CDF)"

> "Если $U \sim U(0,1)$, то $F^{-1}(U)$ имеет распределение с CDF $F$. Это универсальный способ генерации выборки из любого распределения с известной обратной CDF. Метод используется в Monte Carlo симуляциях, генерации синтетических данных, и внутри многих библиотек."

```python
from scipy.stats import expon, weibull_min
import numpy as np

rng = np.random.default_rng(42)

# Экспоненциальное: F^{-1}(u) = -ln(u) / λ
u = rng.uniform(size=1000)
exp_sample = -np.log(u) / lambd

# Произвольное: scipy ppf = percent point function = F^{-1}
u = rng.uniform(size=1000)
sample = weibull_min.ppf(u, c=1.5)
```

**Gotcha:** метод не работает, если CDF не имеет аналитической обратной → используй rejection sampling или MCMC.

## Q16. "Преобразование Бокса-Мюллера — генерация N(0,1)"

> "Из двух независимых $U_1, U_2 \sim U(0,1)$ получаем два независимых $Z_1, Z_2 \sim \mathcal{N}(0,1)$. Классический вопрос на собеседовании: 'как сгенерировать нормальное распределение, имея только генератор равномерных чисел?'"

$$Z_1 = \sqrt{-2\ln U_1} \cos(2\pi U_2), \quad Z_2 = \sqrt{-2\ln U_1} \sin(2\pi U_2)$$

```python
u1, u2 = rng.uniform(size=n), rng.uniform(size=n)
r = np.sqrt(-2 * np.log(u1))
z1 = r * np.cos(2 * np.pi * u2)
z2 = r * np.sin(2 * np.pi * u2)

# Для N(μ, σ²):
x = mu + sigma * z1
```

**Gotcha:** одна пара $(U_1, U_2)$ порождает две нормальные величины — для $n$ значений нужно $\lceil n/2 \rceil$ пар.

## Q17. "Тяжёлые хвосты — как распознать и что с ними делать"

> "Тяжёлые хвосты означают, что экстремальные значения встречаются значительно чаще, чем при нормальном распределении. В реальных данных это частое явление: финансовые доходности, время отклика серверов, размеры файлов. Стандартные методы, основанные на среднем и дисперсии, могут быть ненадёжными."

**Признаки тяжёлых хвостов:**
- Выборочное среднее нестабильно (скачет при увеличении выборки)
- `scipy.stats.kurtosis(data)` значительно > 0 (нормальное = 0)
- QQ-plot отклоняется от прямой на концах

```python
from scipy.stats import kurtosis, probplot

# Эксцесс: > 0 → тяжелее нормального
print(f'Эксцесс: {kurtosis(data):.2f}')

# QQ-plot: отклонения на концах → тяжёлые хвосты
fig, ax = plt.subplots()
probplot(data, dist='norm', plot=ax)
```

**Практические последствия:**

| Распределение | Убывание хвоста | E[X] | Var(X) |
|---------------|----------------|------|--------|
| Нормальное | $\sim e^{-x^2/2}$ | ✓ | ✓ |
| Экспоненциальное | $\sim e^{-\lambda x}$ | ✓ | ✓ |
| Коши | $\sim 1/x^2$ | ✗ | ✗ |
| Парето ($\alpha > 2$) | $\sim 1/x^{\alpha+1}$ | ✓ | ✓ |
| Парето ($1 < \alpha \leq 2$) | $\sim 1/x^{\alpha+1}$ | ✓ | ✗ |

> Тяжёлые хвосты → среднее ненадёжно → используй медиану или trimmed mean. Дисперсия может не существовать → CI на основе CLT не работают.

## Q18. "scipy.stats — краткий справочник"

> "scipy.stats — основная библиотека для работы с распределениями в Python. Единый API для всех распределений."

| Метод | Описание |
|-------|----------|
| `dist.rvs(size=n)` | Генерация выборки |
| `dist.pdf(x)` | Плотность в точке $x$ |
| `dist.cdf(x)` | Функция распределения $F(x)$ |
| `dist.ppf(q)` | Квантиль ($F^{-1}$) |
| `dist.mean()` | Мат. ожидание |
| `dist.var()` | Дисперсия |
| `dist.fit(data)` | MLE параметров по выборке |

```python
from scipy.stats import norm, cauchy, expon, uniform

# Генерация + плотность на одном графике
x = np.linspace(-4, 4, 1000)
sample = norm.rvs(loc=0, scale=1, size=200)
plt.hist(sample, bins=30, density=True, alpha=0.5, label='Гистограмма')
plt.plot(x, norm.pdf(x), label='Плотность N(0,1)')
plt.legend()
```
